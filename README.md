# 🏠 Builder House API

A professional REST API backend built with **Python Flask** and **MariaDB**.

Builder House is a complete backend system with authentication, user management, and admin control features.

---

## 🚀 Features

### Authentication
- ✅ User Registration
- ✅ User Login
- ✅ JWT Authentication
- ✅ Password Hashing (bcrypt)

### User Management
- ✅ View Profile
- ✅ Update Profile
- ✅ User Status Management

### Admin Panel API
- ✅ Admin Middleware
- ✅ Get All Users
- ✅ Get Single User
- ✅ Update User
- ✅ Delete User
- ✅ Change User Role
- ✅ Change User Status

---

## 🛠️ Technology Stack

| Technology | Usage |
|---|---|
| Python | Backend Language |
| Flask | REST API Framework |
| MariaDB | Database |
| JWT | Authentication |
