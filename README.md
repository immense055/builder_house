🚀 Builder House

Professional Blockchain API Platform

Builder House is a modern blockchain API platform built with Flask, Bootstrap 5, and Python. The project is designed to provide developers with a clean dashboard and powerful APIs for blockchain data, wallet integration, and network exploration.

---

✨ Features

- 🔑 API Key Management
- 🪙 Bitcoin Explorer API
- 🌐 Multi-Blockchain Support
- 📊 API Usage Dashboard
- 📖 API Documentation
- ⚡ REST API
- 📱 Responsive Bootstrap UI
- 🔒 Secure Authentication
- 📈 Analytics & Monitoring

🚀 Installation

git clone https://github.com/immense055/builder_house.git
cd builder_house

python -m venv venv
source venv/bin/activate

pip install flask

python app.py

Open your browser:

http://127.0.0.1:5000
